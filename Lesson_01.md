## Работа с Git

И так. Для начала надо инициализировать хранилище. Для этого используется команда 
    
    git init

Пример:

    po@MacBook-Air-Pavel GIT_01 % git init
    Initialized empty Git repository in /Users/po/Desktop/GB/GIT_01/.git/
